# Convolutional Neural Network (CNN)
This repository completes the task of the course **Machine Learning**, which requires to achieve MNIST recognition based on convolutional neural network (CNN) .

## About the report 
You just need to open `tex/main.pdf`, and you can see the report. To see more, please enter `tex/main.tex` to vie the latex code.

## Running Procedure
First install the required packages
``` bash
pip install -r requirements.txt
```
Then run the code
```bash
python main.py
```

To compare the difference between CNN and FNN, you can run the code to compare the accuracy
```bash
python 16.MNIST_recognition_via_FNN.py
```
